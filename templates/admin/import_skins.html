CSV Format

Your CSV file should have the following columns:

  • Name (required)
  • Category (optional, defaults to 'rifle')
  • Price (required)
  • Description (optional)
  • Is Active (optional, defaults to True)

Example:

Name,Category,Price,Description,Is Active
AK-47 | Asiimov,rifle,50.00,A clean and futuristic design,True
Desert Eagle | Blaze,pistol,200.00,Hot and fiery skin,True
